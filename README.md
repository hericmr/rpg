# SantosPunk 2099

Um jogo RPG cyberpunk ambientado em Santos no ano de 2099, desenvolvido com Phaser 3 e TypeScript.

## Sobre o Jogo

Explore o 87º andar da Torre Principal da Corporação Caiçara em um futuro distópico onde Santos está transformada pela tecnologia e mudanças climáticas.

## Características

- Visual retrô inspirado no GameBoy Color
- Sistema de diálogo com NPCs
- Física e colisões
- Câmera que segue o jogador
- Interface nostálgica

## Como Executar

1. Clone o repositório
2. Instale as dependências:
```bash
npm install
```
3. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

## Tecnologias

- Phaser 3
- TypeScript
- Vite (para desenvolvimento)

## Controles

- Setas direcionais: Movimento do personagem
- Espaço: Interagir/Avançar diálogo
- ESC: Fechar diálogo
